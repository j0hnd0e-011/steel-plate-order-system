# Steel Plate Order & Progress Management System

## Overview
Manufacturing order and progress management system built with PHP and MySQL.

This system was developed to replace Excel + Access based operations and improve workflow management.

## Features
- Login authentication
- Order management
- Progress tracking
- Excel import
- User management
- Log management

## Technologies
- PHP
- MySQL
- JavaScript
- HTML/CSS
- Apache (XAMPP)

## Improvements
- Reduced dependency on Excel / Access
- Browser-based operation
- Improved maintainability
- Reduced operational workload
